Training Strategy

Stage 0: Exact Match Injection

50 epochs focusing on simple geometric transformations to build strong spatial understanding.

  • Identity transformations for baseline
  • Pure rotations and reflections
  • Simple translations
  • Combined transformations

Specialized Loss Components

  • Transformation Loss: Penalizes incorrect spatial mappings
  • Discrete Transform Loss: Cross-entropy for rotation/reflection
  • Regularization: Encourages smooth, valid transformations
  • Minimal Residual: 0.05 mix to prevent identity shortcuts

Optimization Details

  • Initialization: Affine matrix starts as identity
  • Learning Rate: 0.01 ‚Üí 0.005 ‚Üí 0.002
  • Batch Size: 512 (effective 2048)
  • Gradient Clipping: 1.0
  • Xavier Initialization: Gain 1.2 for decoder
