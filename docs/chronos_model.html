Key Components

🔄

Bi-LSTM Stack

3-layer bidirectional LSTM with 256 hidden units per direction for capturing temporal dependencies in both directions.

👁️

Temporal Attention

Multi-head attention (8 heads) with learnable position encodings for focusing on relevant temporal patterns.

📊

Pattern Detector

Specialized module for identifying repeating patterns, cycles, and temporal transformations.

🔮

Evolution Predictor

Predicts how patterns evolve over time, learning transformation rules from sequences.

🎯

Spatial Encoder

Encodes each frame independently before temporal processing for robust feature extraction.

🌊

Skip Connections

Direct connections from input to output (0.1 mix) for preserving spatial information.

Temporal Capabilities

  • Pattern Evolution: Tracks how patterns change over time
  • Cyclic Detection: Identifies repeating cycles and periods

  • Sequence Completion: Predicts next frame based on temporal context
  • Motion Tracking: Follows object movement across frames
  • Rule Inference: Learns transformation rules from sequences

Training Strategy

Sequence-Based Training

CHRONOS skips exact match injection and focuses on sequence understanding from the start.

  • Single-frame to multi-frame progression
  • Temporal consistency loss
  • Pattern evolution penalties
  • Sequence prediction tasks

LSTM Optimization

  • Gradient Clipping: Essential for LSTM stability
  • Layer Normalization: Applied after each LSTM layer
  • Dropout: 0.1 for regularization
  • Orthogonal Init: For LSTM weight matrices

Training Details

  • Sequence Length: 1-5 frames progressive
  • Learning Rate: 0.01 → 0.005 → 0.002
  • Batch Size: 512 (effective 2048)
  • Temporal Loss Weight: 0.4
