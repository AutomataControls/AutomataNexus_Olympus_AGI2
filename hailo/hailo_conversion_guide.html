pytorch_to_onnx.py

PyTorch Conversion

Converts trained OLYMPUS PyTorch models (.pt/.pth files) to ONNX format for cross-platform compatibility and Hailo optimization.

  • Supports both .pt and .pth file formats
  • Automatic batch size handling
  • Dynamic input shapes for flexibility
  • Model validation during export
  • ONNX opset version 11 compatibility
  • Constant folding optimization
  • Named input/output tensors
  • Error handling and reporting
üîß

onnx_to_har.py

Hailo Archive Creation

Utilizes Hailo Dataflow Compiler (DFC) in WSL to parse ONNX models and create Hailo Archive (HAR) format for optimization.

  • WSL-based Hailo DFC integration
  • Automatic configuration generation
  • Batch size optimization
  • Model naming conventions
  • DFC validation and error checking
  • Timeout handling for large models
  • Progress reporting
  • Multi-model batch processing
‚ö°

har_to_hef.py

Hailo Executable Format

Compiles HAR models to Hailo Executable Format (HEF) for deployment on Hailo-8 hardware with performance optimization.

  • Hailo-8 hardware targeting
  • High-performance mode compilation
  • Multi-context execution support
  • Memory optimization
  • Performance analysis reporting
  • Compilation time tracking
  • HEF size optimization
  • Throughput metrics

Usage Examples

Step 1: Convert PyTorch models to ONNX
python hailo/scripts/pytorch_to_onnx.py \ --models_dir src/models/checkpoints \ --output_dir hailo/hailo_models/onnx \ --model_names MINERVA ATLAS IRIS CHRONOS PROMETHEUS
Step 2: Parse ONNX to HAR using Hailo DFC
python hailo/scripts/onnx_to_har.py \ --onnx_dir hailo/hailo_models/onnx \ --output_dir hailo/hailo_models/har \ --model_names MINERVA ATLAS IRIS CHRONOS PROMETHEUS
Step 3: Compile HAR to HEF for deployment
python hailo/scripts/har_to_hef.py \ --har_dir hailo/hailo_models/har \ --output_dir hailo/hailo_models/hef \ --model_names MINERVA ATLAS IRIS CHRONOS PROMETHEUS \ --analyze
Complete pipeline execution
# Run all conversion steps ./hailo/scripts/convert_all_models.sh

Requirements & Installation

Prerequisites

  • WSL2 environment with Ubuntu 20.04+
  • Hailo Dataflow Compiler (DFC) installed in WSL
  • Python 3.8+ with PyTorch and ONNX
  • Trained OLYMPUS model checkpoints (.pt/.pth)

Hailo DFC Installation

# In WSL environment wget https://hailo.ai/downloads/dfc/latest/hailo_dataflow_compiler.deb sudo dpkg -i hailo_dataflow_compiler.deb sudo apt-get install -f # Verify installation hailo --version

Python Dependencies

pip install torch torchvision onnx onnxruntime pip install numpy argparse pathlib
