Overview

IRIS specializes in color pattern recognition and transformation within the OLYMPUS AGI2 ensemble. Named after the Greek goddess of the rainbow, IRIS excels at understanding color relationships, mapping transformations, and discovering visual patterns through advanced attention mechanisms.

Key Components

🎨

Color Embeddings

Learnable 64-dimensional embeddings for each of the 10 ARC colors, capturing semantic relationships between colors.

👁️

Color Attention

Multi-head attention mechanism (4 heads) that learns which colors are important for the transformation.

🗺️

Color Mapping Matrix

Learns a 10×10 transformation matrix that defines how each input color maps to output colors.

🔄

Spatial Encoder

Convolutional network that captures spatial color patterns and relationships.

📜

Rule Encoder

LSTM network for learning pattern-based color transformation rules.

✨

Pure Transform

No residual connections - IRIS focuses purely on color transformations.

# IRIS Forward Pass Example
model = EnhancedIrisNet(max_grid_size=30)

# Input: one-hot encoded grid (B, 10, H, W)
input_grid = torch.randn(32, 10, 15, 15)

# Forward pass
outputs = model(input_grid)
predicted_grid = outputs['predicted_output'] # (B, 10, H, W)
color_map = outputs['color_map'] # (B, 10, 10) softmax probabilities
color_attention = outputs['color_attention'] # Attention weights
mapped_output = outputs['mapped_output'] # Direct color mapping result

# Example: Check how color 2 (red) is mapped
red_mapping = color_map[0, 2] # Probabilities for each output color
new_color = red_mapping.argmax() # Most likely output color

Color Transformation Capabilities

  • Color Swapping: Exchange colors (e.g., red ↔ blue)
  • Color Filtering: Keep only specific colors
  • Color Mapping: Complex many-to-one or one-to-many mappings
  • Pattern-based Rules: Context-aware color transformations
  • Color Propagation: Spread colors based on patterns
  • Selective Replacement: Replace colors in specific regions

Training Strategy

Stage 0: Color Focus

Initial training emphasizes pure color transformations with simple patterns.

  • Color swap tasks (2-3 colors)
  • Single color extraction
  • Color fill operations
  • Binary color patterns

Specialized Architecture

  • No Residual: Pure transformation focus
  • Attention Mechanism: Learn color importance
  • Explicit Mapping: 10×10 learnable matrix
  • Spatial Awareness: Context-based decisions

Training Details

  • Xavier Init: Gain 1.5 for strong transforms
  • Biased Init: Varied per color channel
  • Learning Rate: 0.01 → 0.005 → 0.002
  • Color-aware Loss: Penalizes incorrect mappings
