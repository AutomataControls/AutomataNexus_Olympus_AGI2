Training Strategy

VAE-GAN Training

PROMETHEUS skips exact match injection due to its generative nature.

  • Reconstruction loss for accuracy
  • KL divergence for latent regularization
  • Adversarial loss for realism
  • Innovation bonus for novel patterns

Loss Components

  • Reconstruction: L2 + perceptual loss
  • KL Loss: Weight 0.001 for latent regularization
  • Adversarial: WGAN-GP for stable training
  • Innovation: Rewards novel valid patterns
